### Тема
Установка гостевой операционной системы Windows 7 на виртуальную машину, созданную в среде VirtualBox, на хосте с Windows 10.

### Цель работы
- **Теоретическая:** Изучить процесс создания виртуальной машины, настройки её аппаратных ресурсов (ОЗУ, процессор, диск, сеть), установки гостевой ОС и инструментов для интеграции с хост-системой.
- **Практическая:** Приобрести навыки работы с VirtualBox, научиться настраивать виртуальное оборудование, устанавливать Windows 7, подключать образ дополнений гостевой ОС, организовывать обмен файлами между хостом и гостем.

### Задачи
1. Создать новую виртуальную машину в VirtualBox с именем, содержащим фамилию студента.
2. Назначить виртуальной машине 1024 МБ оперативной памяти, 2 ядра процессора с ограничением использования 50%, виртуальный диск VDI динамического формата размером 20 ГБ.
3. Настроить сетевой адаптер в режиме «Сетевой мост» для получения IP-адреса от DHCP хостовой сети.
4. Установить гостевую ОС Windows 7 (32-бит) из ISO-образа, указав имя пользователя (фамилию), имя компьютера (WIN7-VM-фамилия), пароль (P@ssw0rd или пустой).
5. Установить Дополнения гостевой ОС (VirtualBox Guest Additions) для улучшения производительности и интеграции.
6. Проверить работу сети (доступ в интернет), параметры системы (ОЗУ, процессоры) через `msinfo32` и `dxdiag`.
7. Проверить обмен файлами между хостовой Windows 10 и гостевой Windows 7 через общую папку и общий буфер обмена.

### Оборудование и программное обеспечение
- **Программное обеспечение хоста:** Windows 10 (любая редакция) с установленным VirtualBox (версия 6.0 или новее).
- **Программное обеспечение гостя:** ISO-образ установочного диска Windows 7 (32-бит).
- **Оборудование (виртуальное/реальное):** Персональный компьютер с поддержкой аппаратной виртуализации (Intel VT-x / AMD-V), не менее 4 ГБ ОЗУ и 30 ГБ свободного места на диске.

### Краткие теоретические сведения

**Виртуализация** — технология, позволяющая запускать несколько изолированных операционных систем на одном физическом компьютере (хосте). VirtualBox — популярный гипервизор 2-го типа (работает поверх хостовой ОС).

**Ключевые параметры виртуальной машины:**
- **ОЗУ** — выделяется из памяти хоста. Для Windows 7 (32-бит) минимально 512 МБ, рекомендуется 1–2 ГБ.
- **Процессор** — можно выделить несколько виртуальных ядер (до числа физических ядер хоста). Ограничение использования (execution cap) позволяет уменьшить нагрузку на хост.
- **Виртуальный диск** — файл на диске хоста. Тип VDI (родной формат VirtualBox). Динамический диск растёт по мере заполнения данными.
- **Сетевой режим «Сетевой мост» (Bridged Adapter)** — виртуальная машина подключается к той же физической сети, что и хост, получает собственный IP-адрес от DHCP-сервера (роутера) и становится полноправным участником сети. Это позволяет гостевой ОС иметь прямой доступ к интернету и локальным ресурсам.

**Дополнения гостевой ОС (Guest Additions)** — набор драйверов и утилит, улучшающих интеграцию: автоматическое изменение разрешения экрана, общий буфер обмена (shared clipboard), drag-and-drop, синхронизация времени, поддержка общей папки (shared folder). Устанавливаются после установки гостевой ОС.

### Порядок выполнения работы

> **Важно:** По каждому пункту работы необходимо сделать скриншот для последующего включения в отчёт.

#### 1. Подготовительный этап
1. Убедитесь, что VirtualBox установлен на хостовой Windows 10.
2. Скачайте ISO-образ Windows 7 (32-бит) или подготовьте установочный диск.
3. Освободите не менее 20 ГБ свободного места на диске хоста.
4. По желанию можете подготовить USB-флэш носитель установив на него Ventoy. Используйте свою USB-флэш.

#### 2. Основной этап

##### **Задание 1: Создание виртуальной машины**
1. Запустите **VirtualBox**.
2. Нажмите кнопку **Создать**.
3. Укажите параметры:
   - **Имя:** `Win7-VM-<ваша_фамилия>` (например, Win7-VM-Ivanov)
   - **Тип:** Microsoft Windows
   - **Версия:** Windows 7 (32-bit)
4. Назначьте **размер оперативной памяти**: **1024 МБ** (1 ГБ).
5. Создайте виртуальный жесткий диск:
   - Тип файла: **VDI (VirtualBox Disk Image)**
   - Формат хранения: **Динамический виртуальный диск**
   - **Размер:** 20 ГБ
6. Сделайте скриншот окна создания ВМ.

##### **Задание 2: Настройка ресурсов виртуальной машины**
1. Выберите созданную ВМ → **Настроить** (или **Изменить**).
2. Перейдите в раздел **Система** → вкладка **Процессор**:
   - Число CPU: **2** (если хост имеет не менее 2 ядер)
   - **Ограничение использования:** **50%** (ползунок Execution Cap)
3. Перейдите в раздел **Носители**:
   - Выберите пустой привод под контроллером IDE (строка «Пусто»).
   - Нажмите на иконку диска → **Выбрать образ диска** → укажите ISO-файл Windows 7.
4. Перейдите в раздел **Сеть** → Адаптер 1:
   - **Тип подключения:** Сетевой мост (Bridged Adapter)
   - **Имя:** выберите активный сетевой интерфейс вашего хоста (например, Wi-Fi или Ethernet).
5. Сделайте скриншоты настроек (Процессор, Носители, Сеть).

##### **Задание 3: Установка Windows 7**
1. Запустите виртуальную машину (кнопка **Запустить**).
2. Начнётся загрузка с ISO-образа. Дождитесь появления окна установки Windows.
3. Выберите язык, формат времени, раскладку клавиатуры → **Далее** → **Установить**.
4. Примите условия лицензии → **Далее**.
5. Выберите **Полная установка (дополнительные параметры)**.
6. В списке дисков должен отобразиться неразмеченный диск 0 размером 20 ГБ. Выберите его → **Далее** (установка начнётся автоматически).
7. После копирования файлов и перезагрузок настройте параметры:
   - **Имя пользователя:** `<ваша_фамилия>` (строчными буквами)
   - **Имя компьютера:** `WIN7-VM-<ваша_фамилия>` (например, WIN7-VM-Ivanov)
   - **Пароль:** `P@ssw0rd` (или оставьте пустым)
   - **Ключ продукта:** (можно пропустить, нажав «Далее»)
   - **Параметры обновления:** выбрать «Откладывать решение» (или «Не проверять обновления» для ускорения)
   - **Часовой пояс:** выбрать свой
   - **Сеть:** **Общественная сеть**
8. Дождитесь завершения установки и входа в систему. Сделайте скриншот рабочего стола.

##### **Задание 4: Установка Дополнений гостевой ОС (Guest Additions)**
1. В меню VirtualBox выберите **Устройства** → **Подключить образ диска Дополнений гостевой ОС...**.
2. В гостевой Windows 7 откройте **Этот компьютер** (Пуск → Компьютер).
3. В разделе «Устройства со съёмными носителями» появится диск «VirtualBox Guest Additions». Дважды кликните по нему.
4. Запустите `VBoxWindowsAdditions.exe` (если автозапуск не сработал).
5. Следуйте инструкциям мастера установки, оставляя параметры по умолчанию. После установки потребуется перезагрузка (нажмите «Перезагрузить сейчас»).
6. Сделайте скриншот процесса установки или сообщения об успешной установке.

##### **Задание 5: Проверка работы системы**
1. **Проверка интернета:** запустите браузер (Internet Explorer) → откройте любой сайт (например, `ya.ru`). Сделайте скриншот.
2. **Проверка параметров системы:**
   - Нажмите `Win + R`, введите `msinfo32`, нажмите Enter.
   - Проверьте: **Объём оперативной памяти (ОЗУ)** – должно быть 1024 МБ; **Процессор(ы)** – должно быть 2 ядра.
   - Сделайте скриншот окна `msinfo32`.
   - Выполните `dxdiag` → вкладка **Система** – проверьте версию DirectX, ОЗУ, процессор. Сделайте скриншот.
3. **Проверка обмена файлами:**
   - **Способ 1: Общая папка (Shared Folder)**
     - В меню VirtualBox → **Устройства** → **Общие папки** → **Настройки общих папок**.
     - Добавьте новую общую папку: укажите каталог на хосте (например, `C:\Share`), выберите **Автомонтировать**, **Постоянная**.
     - В гостевой Windows 7 откройте **Этот компьютер** → раздел «Сетевое размещение» или выполните `\\vboxsvr\sharename`.
     - Скопируйте текстовый файл с хоста в гостевую и обратно. Сделайте скриншот.
   - **Способ 2: Общий буфер обмена (Shared Clipboard)**
     - Убедитесь, что в меню VirtualBox → **Устройства** → **Общий буфер обмена** выбран режим **Двунаправленный**.
     - Создайте текст в Блокноте на хосте, скопируйте (Ctrl+C), переключитесь в гостевую ОС, вставьте (Ctrl+V) в Блокнот. Сделайте скриншот.

##### **Задание 6 (дополнительное): Создание снапшота (снимка состояния)**
1. Выключите виртуальную машину (Пуск → Завершение работы).
2. В VirtualBox выберите ВМ → **Снимки** → **Создать**.
3. Укажите имя «До установки ПО» и описание.
4. Сделайте скриншот окна с созданным снапшотом.

#### 3. Контрольный этап
1. Убедитесь, что все шаги выполнены, скриншоты сохранены.
2. Ответьте на контрольные вопросы (письменно).
3. При необходимости восстановите виртуальную машину из снапшота.

### Контрольные вопросы
1. Почему для Windows 7 выбран режим сети «Сетевой мост»? (обеспечивает прямой доступ к сети хоста и интернету, виртуальная машина получает собственный IP, что позволяет тестировать сетевые службы и не требует настройки NAT)
2. Как перенести файлы с хостовой ОС на гостевую Windows 7? (через общую папку, через общий буфер обмена, через drag-and-drop, через USB-накопитель или по сети)
3. Для чего нужны «Дополнения гостевой ОС»? (улучшают интеграцию: автоматическое разрешение экрана, общий буфер обмена, общие папки, синхронизацию времени, поддержку мыши без захвата)
4. Как создать снапшот (снимок состояния) ВМ перед установкой ПО? (выключить ВМ, в главном окне VirtualBox выбрать ВМ → «Снимки» → «Создать»)
