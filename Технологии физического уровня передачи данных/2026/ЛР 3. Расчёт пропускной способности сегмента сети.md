## **Теоретические сведения**

Для передачи цифрового сигнала по витой паре используется многопозиционная амплитудно-фазовая модуляция (например, PAM-16, PAM-32). Максимальная пропускная способность канала (в бит/с) ограничена теоремой Шеннона – Хартли:

$$ C = B \cdot \log_2\left(1 + \text{SNR}\right) $$

где:
- $C$ – пропускная способность (бит/с);
- $B$ – полоса частот, занимаемая сигналом (Гц);
- ${SNR}$ – отношение сигнал/шум на входе приёмника (по мощности, в разах).

В реальных кабельных системах основными помехами являются:
- **Собственное затухание** (ослабление полезного сигнала с частотой и длиной).
- **Перекрёстные наводки NEXT** (Near End Cross Talk) – помеха от передатчика, работающего на том же конце линии.

Результирующее отношение сигнал/шум (в разах) можно оценить как:

$$ \text{SNR} = \frac{P_{\text{вых}}}{P_{\text{NEXT}}} $$

где $P_{\text{вых}}$ – мощность полезного сигнала на входе приёмника, $P_{\text{NEXT}}$ – мощность перекрёстной помехи на том же конце.

В децибелах:

$$
\text{SNR}_{\text{дБ}} = P_{\text{вых,дБм}} - P_{\text{NEXT,дБм}} = (\text{PS}_{\text{дБм}} - A_{\text{дБ}}) - \text{NEXT}_{\text{дБ}}
$$

где $\text{PS}$ – мощность передатчика (обычно от 0 до +10 дБм), $A$ – затухание кабеля на данной длине и частоте, $\text{NEXT}$ – коэффициент перекрёстной наводки на ближнем конце (в дБ, задаётся для кабеля).

Для упрощения в работе используется **ACS** (Attenuation to Crosstalk Ratio) – отношение затухания к перекрёстной наводке:

$$
\text{ACS}_{\text{дБ}} = \text{NEXT}_{\text{дБ}} - A_{\text{дБ}}
$$

Этот параметр характеризует помехозащищённость канала. Для надёжной работы ACS должен быть не менее 10–15 дБ (в зависимости от уровня кодирования).

- NEXT (на 100 м) = 30 дБ (справочное значение для Cat5e).

### **3. Влияние длины кабеля на пропускную способность**
Известно, что затухание пропорционально длине: $A(L) = a \cdot L$, а NEXT (дБ) для кабеля ухудшается на 15·log10(L) относительно значения на 100 м.

Для кабеля Cat5e:
- $a = 0.24 \, \text{дБ/м}$
- $\text{NEXT}_{100} = 30 \, \text{дБ}$

Заполните таблицу для длин \(L = 50, 100, 150, 200\) м:

| Длина, м | Затухание A, дБ | NEXT, дБ | ACS, дБ (NEXT - A) | SNR, дБ (при Pпер=4 дБм) | Пропускная способность C, Мбит/с (по Шеннону, B=100 МГц) |
| -------- | --------------- | -------- | ------------------ | ------------------------ | -------------------------------------------------------- |
| 50       |                 |          |                    |                          |                                                          |
| 100      |                 |          |                    |                          |                                                          |
| 150      |                 |          |                    |                          |                                                          |
| 200      |                 |          |                    |                          |                                                          |

*Указание: NEXT(L) = NEXT_100 - 15·log10(L/100).*  
*SNR_дБ = P_пер - A(L) - (P_пер - NEXT(L)) = NEXT(L) - A(L) = ACS_дБ* (при условии, что уровень помехи не зависит от мощности передатчика).  
Таким образом, SNR_дБ численно равен ACS_дБ для ближнего конца.
