### **Краткие теоретические сведения**

**Магические методы** (или dunder-методы — от double underscore) — это специальные методы в Python, которые начинаются и заканчиваются двумя подчёркиваниями (например, `__init__`). Они позволяют определить поведение объектов для встроенных операций.

**Основные группы магических методов:**

**1. Инициализация и представление:**
- `__init__(self, ...)` – конструктор, вызывается при создании объекта
- `__str__(self)` – вызывается при преобразовании объекта в строку (print, str())
- `__repr__(self)` – вызывается при отображении объекта в интерпретаторе, должен возвращать "официальное" строковое представление

**2. Операторы сравнения:**
- `__eq__(self, other)` – равенство (`==`)
- `__ne__(self, other)` – неравенство (`!=`)
- `__lt__(self, other)` – меньше (`<`)
- `__le__(self, other)` – меньше или равно (`<=`)
- `__gt__(self, other)` – больше (`>`)
- `__ge__(self, other)` – больше или равно (`>=`)

**3. Арифметические операторы:**
- `__add__(self, other)` – сложение (`+`)
- `__sub__(self, other)` – вычитание (`-`)
- `__mul__(self, other)` – умножение (`*`)
- `__truediv__(self, other)` – деление (`/`)
- `__floordiv__(self, other)` – целочисленное деление (`//`)

**4. Операторы доступа к элементам:**
- `__len__(self)` – длина объекта (`len()`)
- `__getitem__(self, key)` – доступ по индексу или ключу (`obj[key]`)
- `__setitem__(self, key, value)` – присваивание по индексу (`obj[key] = value`)
- `__delitem__(self, key)` – удаление элемента (`del obj[key]`)

**5. Итерация:**
- `__iter__(self)` – возвращает итератор для объекта
- `__next__(self)` – возвращает следующий элемент в итераторе

**Пример реализации магических методов:**
```python
class Vector:
    def __init__(self, x, y):
        self.x = x
        self.y = y
    
    def __str__(self):
        return f"Vector({self.x}, {self.y})"
    
    def __repr__(self):
        return f"Vector({self.x}, {self.y})"
    
    def __add__(self, other):
        return Vector(self.x + other.x, self.y + other.y)
    
    def __eq__(self, other):
        return self.x == other.x and self.y == other.y
    
    def __len__(self):
        return 2  # Вектор всегда имеет 2 координаты
    
    def __getitem__(self, index):
        if index == 0:
            return self.x
        elif index == 1:
            return self.y
        else:
            raise IndexError("Индекс вне диапазона")
```

```python
class Money:
    """Класс, представляющий денежную сумму в рублях и копейках."""
    
    def __init__(self, rubles, kopecks):
        self.rubles = rubles
        self.kopecks = kopecks
        self._normalize()  # Приводим к нормальному виду
    
    def _normalize(self):
        """Приводит сумму к нормальному виду (копейки от 0 до 99)."""
        if self.kopecks >= 100:
            self.rubles += self.kopecks // 100
            self.kopecks = self.kopecks % 100
        elif self.kopecks < 0:
            # Упрощённая обработка отрицательных копеек
            self.rubles += self.kopecks // 100 - 1
            self.kopecks = 100 + (self.kopecks % 100)
    
    # Строковое представление
    def __str__(self):
        """Вызывается при print() и str()."""
        return f"{self.rubles} руб. {self.kopecks:02d} коп."
    
    def __repr__(self):
        """Вызывается в интерпретаторе."""
        return f"Money({self.rubles}, {self.kopecks})"
    
    # Операторы сравнения
    def __eq__(self, other):
        """Проверка на равенство."""
        if isinstance(other, Money):
            return (self.rubles == other.rubles and 
                    self.kopecks == other.kopecks)
        return False
    
    def __lt__(self, other):
        """Меньше."""
        if isinstance(other, Money):
            self_total = self.rubles * 100 + self.kopecks
            other_total = other.rubles * 100 + other.kopecks
            return self_total < other_total
        return NotImplemented
    
    # Арифметические операторы
    def __add__(self, other):
        """Сложение."""
        if isinstance(other, Money):
            total_kopecks = (self.rubles * 100 + self.kopecks) + \
                           (other.rubles * 100 + other.kopecks)
            return Money(total_kopecks // 100, total_kopecks % 100)
        return NotImplemented
    
    def __sub__(self, other):
        """Вычитание."""
        if isinstance(other, Money):
            total_kopecks = (self.rubles * 100 + self.kopecks) - \
                           (other.rubles * 100 + other.kopecks)
            if total_kopecks < 0:
                return Money(total_kopecks // 100, total_kopecks % 100)
            return Money(total_kopecks // 100, total_kopecks % 100)
        return NotImplemented
    
    def __mul__(self, factor):
        """Умножение на число."""
        if isinstance(factor, (int, float)):
            total_kopecks = (self.rubles * 100 + self.kopecks) * factor
            return Money(int(total_kopecks // 100), int(total_kopecks % 100))
        return NotImplemented

print("Создаём объекты:")
m1 = Money(10, 50)
m2 = Money(5, 75)
m3 = Money(10, 50)

print(f"m1 = {m1}")          # __str__
print(f"m2 = {m2}")
print(f"m3 = {m3}")

print("\nСравнение:")
print(f"m1 == m3: {m1 == m3}")   # __eq__
print(f"m1 == m2: {m1 == m2}")
print(f"m1 > m2: {m1 > m2}")     # __gt__ автоматически генерируется из __lt__
print(f"m1 < m2: {m1 < m2}")     # __lt__

print("\nАрифметика:")
print(f"m1 + m2 = {m1 + m2}")    # __add__
print(f"m1 - m2 = {m1 - m2}")    # __sub__
print(f"m1 * 3 = {m1 * 3}")      # __mul__

print("\nОтрицательные суммы:")
m4 = Money(0, 50) - Money(1, 0)
print(f"0.50 - 1.00 = {m4}")     # Отрицательные рубли
```
