### **Краткие теоретические сведения**

**Наследование** – механизм, позволяющий создать новый класс на основе существующего. Новый класс (производный, дочерний) получает все атрибуты и методы базового (родительского) класса и может расширять или изменять их.

```python
class BaseClass:
    def method(self):
        print("Метод базового класса")

class DerivedClass(BaseClass):
    def method(self):          # переопределение метода
        print("Метод производного класса")
```

**Полиморфизм** – способность объектов с одинаковым интерфейсом иметь различную реализацию. В Python полиморфизм проявляется, когда мы вызываем один и тот же метод у объектов разных классов, и каждый объект выполняет свою версию метода.

**Абстрактный метод** – метод базового класса, который не выполняет полезных действий, а служит лишь шаблоном. В Python его можно создать с помощью `pass` или выбросить исключение `NotImplementedError`. В данной работе мы используем простой `pass` или заглушку, возвращающую ноль/пустую строку, чтобы показать намерение переопределить метод в потомке.

##### **Задание 1. Общий пример (выполняется всеми студентами)**
Изучите приведённый ниже код, реализующий простую иерархию классов «Животные». Скопируйте его в свой файл. Обратите внимание на то, как базовый класс `Animal` определяет методы-заглушки, а классы `Dog` и `Cat` переопределяют их. Затем создайте список из разных животных и вызовите их методы в цикле, чтобы увидеть полиморфизм в действии.

```python
# Базовый класс
class Animal:
    def __init__(self, name):
        self.name = name

    def speak(self):
        # Заглушка, будет переопределена
        pass

    def move(self):
        # Заглушка
        pass

    def info(self):
        print(f"Я животное по имени {self.name}")

# Класс-наследник Dog
class Dog(Animal):
    def speak(self):
        print(f"{self.name} говорит: Гав-гав!")

    def move(self):
        print(f"{self.name} бегает на четырёх лапах")

    def info(self):
        print(f"Я собака по имени {self.name}")

# Класс-наследник Cat
class Cat(Animal):
    def speak(self):
        print(f"{self.name} говорит: Мяу-мяу!")

    def move(self):
        print(f"{self.name} крадётся бесшумно")

    def info(self):
        print(f"Я кошка по имени {self.name}")

# Демонстрация полиморфизма
animals = [Dog("Шарик"), Cat("Мурка"), Dog("Бобик")]

for animal in animals:
    animal.info()
    animal.speak()
    animal.move()
    print("-" * 20)
```

**Что нужно сделать:** Запустите этот код, убедитесь, что каждый объект ведёт себя по-своему. Попробуйте добавить ещё один класс-наследник (например, `Bird`) с собственной реализацией методов.

**Вариант 1. Геометрические фигуры**
Базовый класс `Shape`:
- метод `area()` – возвращает 0 (заглушка)

- метод `info()` – выводит строку "Это геометрическая фигура"
