### **Краткие теоретические сведения**

**Модуль `os`** предоставляет функции для взаимодействия с операционной системой:
- `os.getcwd()` – текущая рабочая директория.
- `os.listdir(path)` – список файлов и папок в указанной директории.
- `os.path.join(path, *paths)` – объединение путей с учётом разделителя ОС.
- `os.path.isfile(path)`, `os.path.isdir(path)` – проверка типа.
- `os.path.getsize(path)` – размер файла в байтах.
- `os.path.getctime(path)`, `os.path.getmtime(path)` – время создания и последнего изменения (timestamp).

**Модуль `sys`** даёт доступ к некоторым переменным и функциям интерпретатора:
- `sys.argv` – список аргументов командной строки (первый элемент – имя скрипта).
- `sys.exit()` – завершение программы с кодом возврата.

**Модуль `datetime`** используется для работы с датами и временем:
- `datetime.datetime.fromtimestamp(timestamp)` – преобразование timestamp в объект datetime.
- Методы `strftime()` для форматирования даты в строку.
- Текущее время: `datetime.datetime.now()`.

**Модуль `json`** позволяет сохранять структуры данных Python в текстовый формат JSON и загружать обратно:
- `json.dump(obj, file)` – запись объекта в файл.
- `json.load(file)` – загрузка объекта из файла.
- `json.dumps()` – сериализация в строку.

**Пример работы с модулями:**
```python
import os
import sys
import json
from datetime import datetime


def audit_dir(dir):
    result = []

    for item in os.listdir(dir):
        full_path = os.path.join(dir, item)
        if os.path.isfile(full_path):
            file_stat = os.stat(full_path)
            str_time = datetime.fromtimestamp(file_stat.st_mtime)
            info = {
                "name": item,
                "size": file_stat.st_size,
                "time": str(str_time)
            }
            result.append(info)

    return result


def to_json(stat):
    with open("result.json", "w") as f:
        json.dump(stat, f, indent=4)


if __name__ == "__main__":
    if len(sys.argv) != 2:
        print("Необходимо указать путь к файлу")
        sys.exit(1)

    dir = sys.argv[1]

    if not os.path.isdir(dir):
        print("В аргументе указана не дириктория!")
        sys.exit(1)

    res = audit_dir(dir)

    to_json(res)
```
