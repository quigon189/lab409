### **Краткие теоретические сведения**

**Класс** – это шаблон для создания объектов, который объединяет данные (атрибуты) и функции (методы) для работы с ними. Объект – конкретный экземпляр класса.

**Определение класса:**
```python
class ИмяКласса:
    def __init__(self, параметры):
        self.атрибут = значение
    def метод(self):
        # тело метода
```

**Конструктор `__init__`** вызывается автоматически при создании объекта. Параметр `self` ссылается на сам создаваемый объект.

**Инкапсуляция** – скрытие внутреннего состояния объекта. В Python нет строгих модификаторов доступа, но существуют соглашения:
- `публичный` атрибут (public) – доступен напрямую.
- `_защищённый` (protected) – одно подчёркивание в начале. По соглашению, не должен использоваться вне класса и его наследников.
- `__приватный` (private) – два подчёркивания в начале. Интерпретатор изменяет имя атрибута (name mangling), чтобы затруднить доступ извне.

**Геттеры и сеттеры** нужны для управления доступом к атрибутам с возможной валидацией. В Python для этого используют декоратор `@property`:
```python
class Person:
    def __init__(self, name, age):
        self._name = name
        self._age = age

    @property
    def age(self):
        return self._age

    @age.setter
    def age(self, value):
        if value < 0:
            raise ValueError("Возраст не может быть отрицательным")
        self._age = value
```
Теперь к `age` можно обращаться как к атрибуту (`obj.age = 25`), но при этом выполняется проверка.

##### **Задание 1. Общий пример (выполняется всеми студентами)**
Изучите приведённый ниже класс `Student`. Скопируйте его в свой файл. Создайте несколько объектов этого класса и проверьте работу методов.

```python
class Student:
    """Класс, представляющий студента с именем, возрастом и средней оценкой."""
    
    def __init__(self, name, age, average_grade):
        self.name = name                # публичный атрибут
        self._age = age                  # защищённый атрибут
        self.__average_grade = average_grade  # приватный атрибут

    # Геттер для возраста (защищённого поля) - через метод
    def get_age(self):
        return self._age

    # Сеттер для возраста с проверкой
    def set_age(self, new_age):
        if new_age <= 0:
            print("Возраст должен быть положительным!")
        else:
            self._age = new_age

    # Свойство для среднего балла (приватное поле)
    @property
    def average_grade(self):
        return self.__average_grade

    @average_grade.setter
    def average_grade(self, value):
        if 0 <= value <= 10:
            self.__average_grade = value
        else:
            print("Оценка должна быть от 0 до 10!")

    # Метод для вывода информации о студенте
    def display_info(self):
        print(f"Студент: {self.name}, возраст: {self._age}, средний балл: {self.__average_grade}")

# Создание объектов
student1 = Student("Иван", 20, 8.5)
student2 = Student("Мария", 19, 9.2)

# Проверка доступа
print(student1.name)          # публичное поле доступно
print(student1._age)           # защищённое поле технически доступно, но так делать не рекомендуется
# print(student1.__average_grade)  # Ошибка! AttributeError

# Использование методов
student1.display_info()
student1.set_age(21)
print(student1.get_age())

# Использование свойств
print(student1.average_grade)
student1.average_grade = 9.0
student1.display_info()
```
