### **Краткие теоретические сведения**

**Файл** — это именованная область данных на носителе информации. В Python работа с файлами осуществляется через файловые объекты.

**Основные режимы открытия файла:**
- `'r'` — чтение (режим по умолчанию)
- `'w'` — запись (создает файл или перезаписывает существующий)
- `'a'` — добавление в конец файла
- `'x'` — создание нового файла (ошибка, если файл существует)
- `'t'` — текстовый режим (по умолчанию)
- `'b'` — бинарный режим

**Контекстный менеджер `with`** — лучший способ работы с файлами, так как автоматически закрывает файл после завершения блока кода, даже если произошла ошибка.

```python
with open('file.txt', 'r', encoding='utf-8') as file:
    data = file.read()
# Файл автоматически закрывается здесь
```

**Основные методы работы с текстовыми файлами:**
- `read()` — чтение всего файла
- `readline()` — чтение одной строки
- `readlines()` — чтение всех строк в список
- `write(text)` — запись текста в файл
- `writelines(list)` — запись списка строк в файл

**Кодировка `utf-8`** используется для корректной работы с русскими буквами и специальными символами.

def create_stats_file(filename, data):
    """Создает файл и записывает в него статистические данные."""
    with open(filename, 'w', encoding='utf-8') as file:
        file.write("Статистика измерений\n")
        file.write("=" * 25 + "\n")
        for item in data:
            file.write(f"{item}\n")
    print(f"Файл '{filename}' успешно создан.")

# Пример использования
measurements = [15.2, 18.5, 22.1, 19.8, 17.3]

**Пример 2: Чтение и обработка файла**
```python
def read_and_analyze(filename):
    """Читает числа из файла и вычисляет статистику."""
    try:
        with open(filename, 'r', encoding='utf-8') as file:
            lines = file.readlines()
        
        # Пропускаем заголовки и преобразуем строки в числа
        numbers = []
        for line in lines:
            line = line.strip()
            if line and line[0].isdigit():  # Проверяем, начинается ли строка с цифры
                try:
                    numbers.append(float(line))
                except ValueError:
                    continue
        
        if not numbers:
            return "Нет числовых данных для анализа"
        
        # Вычисляем статистику
        stats = {
            'количество': len(numbers),
            'сумма': sum(numbers),
            'среднее': sum(numbers) / len(numbers),
            'максимум': max(numbers),
            'минимум': min(numbers)
        }
        return stats
    
    except FileNotFoundError:
        return f"Файл '{filename}' не найден"

# Пример использования
result = read_and_analyze('measurements.txt')
if isinstance(result, dict):
    print("Результаты анализа:")
    for key, value in result.items():
        print(f"{key}: {value}")
else:
    print(result)
```
