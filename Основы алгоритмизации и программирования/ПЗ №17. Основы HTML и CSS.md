### **Тема**
Создание статических веб-страниц с использованием HTML и фреймворка Bootstrap. Структура HTML-документа, подключение стилей, адаптивная сетка, навигация между страницами.

### **Цель работы**
- **Теоретическая:** Изучить базовую структуру HTML-документа, назначение основных тегов, принципы работы CSS-фреймворка Bootstrap для быстрой адаптивной вёрстки.
- **Практическая:** Приобрести навыки создания многостраничного сайта с навигацией, использованием сетки Bootstrap, карточек, форм, а также самостоятельного подбора изображений для контента.

### **Задачи**
1. Освоить структуру HTML-документа (скелет страницы) и роль каждого элемента.
2. Научиться подключать Bootstrap через CDN и использовать его классы.
3. Реализовать навигационное меню для перехода между страницами.
4. Создать не менее двух связанных страниц (главная, о себе/о компании, контакты/форма).
5. Использовать компоненты Bootstrap: навигационную панель, сетку, карточки, кнопки, форму.
6. Самостоятельно подобрать тематические изображения для контента.

### **Оборудование и программное обеспечение**
- **Программное обеспечение:** Любой текстовый редактор (VS Code, Notepad++, Sublime Text) или среда разработки; веб-браузер (Chrome, Firefox, Edge) с инструментами разработчика (F12).
- **Оборудование:** Персональный компьютер с доступом в интернет (для подключения Bootstrap CDN и поиска изображений).

### **Краткие теоретические сведения**

#### **Структура HTML-документа (скелет сайта)**

Любая HTML-страница имеет обязательные элементы, которые образуют её «скелет». Рассмотрим каждый из них.

```html
<!DOCTYPE html>                <!-- 1. Объявление типа документа (HTML5) -->
<html lang="ru">               <!-- 2. Корневой элемент, атрибут lang указывает язык -->
<head>                         <!-- 3. Голова страницы – метаданные, не отображается -->
    <meta charset="UTF-8">     <!-- Кодировка символов (русские буквы) -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                               <!-- Адаптив для мобильных устройств -->
    <title>Название вкладки</title>   <!-- Заголовок страницы (во вкладке браузера) -->
    <link rel="stylesheet" href="..."> <!-- Подключение внешних стилей -->
</head>
<body>                         <!-- 4. Тело страницы – видимое содержимое -->
    <h1>Заголовок</h1>
    <p>Текст параграфа</p>
    <!-- Здесь будет весь видимый контент -->
</body>
</html>                        <!-- 5. Закрывающий тег html -->
```

**Пояснение частей:**
- `<!DOCTYPE html>` – сообщает браузеру, что используется современный стандарт HTML5. Без этого документа страница может отображаться в «режиме совместимости» с искажениями.
- `<html lang="ru">` – корневой тег; атрибут `lang` помогает поисковикам и программам чтения с экрана определить язык содержимого.
- `<head>` – содержит служебную информацию: кодировку, заголовок, подключения CSS и скриптов, мета-теги для SEO и соцсетей. Ничего из этого не видно на самой странице.
- `<meta charset="UTF-8">` – задаёт кодировку UTF-8, которая поддерживает все символы, включая русские буквы, знаки препинания, эмодзи.
- `<meta name="viewport">` – управляет масштабированием на мобильных устройствах; делает сайт адаптивным.
- `<title>` – текст, который отображается на вкладке браузера и используется поисковиками.
- `<link>` – подключает внешние ресурсы: CSS-файлы (стили), иконки, шрифты.
- `<body>` – всё, что написано внутри, будет отображено на странице.

#### **Bootstrap – фреймворк для быстрой вёрстки**

Bootstrap – это набор готовых CSS-классов и JavaScript-компонентов. Чтобы его использовать, достаточно подключить CDN-ссылки в разделе `<head>` (стили) и перед закрывающим `</body>` (скрипты). Пример подключения (Bootstrap 5):

```html
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
```

**Основные классы Bootstrap:**
- `.container` – фиксированный контейнер (отступы по бокам, центрирование). `.container-fluid` – на всю ширину экрана.
- `.row` – ряд, в котором размещаются колонки.
- `.col-*` – колонки. Сетка Bootstrap имеет 12 колонок. Например, `.col-md-6` занимает половину ширины на средних экранах (от 768px) и растягивается на маленьких.
- `.navbar` – навигационная панель.
- `.card` – карточка (блок с рамкой, заголовком, текстом, кнопкой).
- `.btn`, `.btn-primary` – кнопки с оформлением.
- `.form-control` – стилизованное поле ввода.

**Почему Bootstrap удобен:** не нужно писать свои CSS-правила для сетки, адаптивности и типовых компонентов; всё уже оттестировано в разных браузерах.

#### **2. Основной этап**

##### **Шаг 1. Создание базового шаблона (скелета) страницы**

Создайте файл `index.html` и напишите в нём следующий код с подробными комментариями:

```html
<!DOCTYPE html>
<html lang="ru">
<head>
    <!-- Кодировка для корректного отображения русских букв -->
    <meta charset="UTF-8">
    <!-- Адаптив: ширина равна ширине экрана, масштаб 100% -->
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <!-- Заголовок, отображаемый на вкладке браузера -->
    <title>Мой сайт на Bootstrap</title>
    <!-- Подключение Bootstrap CSS (стили) -->
    <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">
    <!-- Здесь можно добавить свои стили (в теге style или отдельном файле) -->
    <style>
        /* Небольшие собственные стили (например, фон, отступы) */
        body {
            background-color: #f5f5f5;
        }
        footer {
            text-align: center;
            padding: 20px;
            margin-top: 40px;
            background-color: #e9ecef;
        }
    </style>
</head>
<body>

    <!-- НАВИГАЦИОННАЯ ПАНЕЛЬ (будет на всех страницах) -->
    <nav class="navbar navbar-expand-lg bg-body-tertiary">
        <div class="container-fluid">
            <!-- Логотип или название сайта -->
            <a class="navbar-brand" href="index.html">Моя тема</a>
            <!-- Кнопка для мобильных (гамбургер) -->
            <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarNav">
                <span class="navbar-toggler-icon"></span>
            </button>
            <!-- Ссылки, которые сворачиваются на мобильных -->
            <div class="collapse navbar-collapse" id="navbarNav">
                <ul class="navbar-nav">
                    <li class="nav-item"><a class="nav-link" href="index.html">Главная</a></li>
                    <li class="nav-item"><a class="nav-link" href="about.html">О нас</a></li>
                    <li class="nav-item"><a class="nav-link" href="contacts.html">Контакты</a></li>
                </ul>
            </div>
        </div>
    </nav>

    <!-- ОСНОВНОЕ СОДЕРЖИМОЕ (будет разным на каждой странице) -->
    <div class="container my-4">  <!-- my-4 – вертикальные отступы (margin top+bottom) -->
        <!-- Здесь будет уникальное содержимое каждой страницы -->
        <h1>Главная страница</h1>
        <p>Добро пожаловать!</p>
    </div>

    <!-- ПОДВАЛ (общий для всех страниц) -->
    <footer>
        <p>&copy; 2025 Моя первая вёрстка на Bootstrap</p>
    </footer>

    <!-- Подключение JavaScript Bootstrap (для работы анимаций, гамбургера) -->
    <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
```

**Что здесь важно:**
- Навигационная панель и подвал вынесены на каждой странице одинаково.
- Добавлен тег `<style>` для собственных мелких правок (фон подвала, цвет фона body). Это необязательно, но полезно.
- Использован контейнер `.container` для центрирования и отступов.
- Класс `my-4` – сокращение для `margin-top: 1.5rem` и `margin-bottom: 1.5rem`.

##### **Шаг 2. Создание содержимого главной страницы (index.html)**

Замените блок `<div class="container my-4">...</div>` на следующий код, который использует сетку Bootstrap и карточки. **Вместо картинки-заглушки студенты должны вставить реальное изображение по своей теме.** Для этого:
- Найдите в интернете любую картинку, сохраните её в ту же папку или используйте прямую ссылку на изображение (URL).
- В атрибуте `src` укажите путь к вашему изображению. Если изображение лежит в папке, просто `images/файл.jpg` (создайте папку `images` и поместите туда картинки). Можно также использовать прямую ссылку из интернета, но убедитесь, что она доступна.

**Пример для главной страницы (тема «Космос»):**

```html
<div class="container my-4">
    <h1 class="text-center mb-4">Добро пожаловать в мир космоса!</h1>
    <!-- Ряд с тремя колонками на больших экранах и одной на мобильных -->
    <div class="row">
        <div class="col-md-4 mb-3">
            <div class="card h-100">
                <!-- Вместо placeholder – реальное изображение -->
                <img src="images/planet.jpg" class="card-img-top" alt="Планета">
                <div class="card-body">
                    <h5 class="card-title">Планеты</h5>
                    <p class="card-text">Узнайте интересные факты о планетах Солнечной системы.</p>
                    <a href="about.html" class="btn btn-primary">Подробнее</a>
                </div>
            </div>
        </div>
        <div class="col-md-4 mb-3">
            <div class="card h-100">
                <img src="images/star.jpg" class="card-img-top" alt="Звезда">
                <div class="card-body">
                    <h5 class="card-title">Звёзды</h5>
                    <p class="card-text">Как рождаются и умирают звёзды? Читайте у нас.</p>
                    <a href="about.html" class="btn btn-primary">Подробнее</a>
                </div>
            </div>
        </div>
        <div class="col-md-4 mb-3">
            <div class="card h-100">
                <img src="images/galaxy.jpg" class="card-img-top" alt="Галактика">
                <div class="card-body">
                    <h5 class="card-title">Галактики</h5>
                    <p class="card-text">Миллиарды галактик и тайны вселенной.</p>
                    <a href="contacts.html" class="btn btn-primary">Связаться</a>
                </div>
            </div>
        </div>
    </div>
</div>
```

**Пояснения:**
- `text-center` – выравнивание текста по центру.
- `mb-4` – отступ снизу (margin-bottom).
- `row` – ряд.
- `col-md-4` – каждая колонка занимает 4 из 12 возможных (т.е. 3 колонки в ряд) на экранах шириной ≥768px; на маленьких экранах колонки будут одна под другой.
- `mb-3` – отступ снизу для карточек, чтобы между ними был промежуток.
- `card h-100` – карточки одной высоты (выравнивание по высоте ряда).

##### **Шаг 3. Страница «О нас» (about.html)**

Скопируйте скелет из index.html (от `<!DOCTYPE>` до `</html>`), измените заголовок `<title>` и содержимое контейнера. Здесь используйте сетку 2 колонки: одна для текста, вторая для изображения.

```html
<div class="container my-4">
    <div class="row align-items-center">
        <div class="col-md-6">
            <h2>О нашем проекте</h2>
            <p>Мы рассказываем о космосе простыми словами. Наша цель – вдохновить людей на изучение вселенной.</p>
            <p>Здесь вы найдёте удивительные факты, красивые фотографии от телескопов и новости астрономии.</p>
            <ul class="list-group list-group-flush">
                <li class="list-group-item">📡 Более 100 статей</li>
                <li class="list-group-item">🪐 Фотографии NASA</li>
                <li class="list-group-item">🔭 Советы начинающим астрономам</li>
            </ul>
        </div>
        <div class="col-md-6">
            <!-- Реальное изображение по теме (например, телескоп или астронавт) -->
            <img src="images/telescope.jpg" class="img-fluid rounded" alt="Телескоп">
        </div>
    </div>
</div>
```

**Пояснения:**
- `align-items-center` – вертикальное выравнивание по центру.
- `list-group` – компонент Bootstrap для стилизации списков.
- `img-fluid` – делает изображение адаптивным (масштабируется).
- `rounded` – скругляет углы.

##### **Шаг 4. Страница «Контакты» (contacts.html)**

На странице контактов разместите контактную информацию и форму обратной связи (без реальной отправки). Форму также стилизуем Bootstrap.

```html
<div class="container my-4">
    <div class="row">
        <div class="col-md-5">
            <h2>Наши контакты</h2>
            <p><strong>Адрес:</strong> г. Москва, ул. Космическая, д. 42</p>
            <p><strong>Телефон:</strong> +7 (123) 456-78-90</p>
            <p><strong>Email:</strong> info@spaceworld.ru</p>
            <hr>
            <h5>Мы в соцсетях</h5>
            <p>VK, Telegram, YouTube</p>
            <!-- Можно добавить иконки (FontAwesome), но это не обязательно -->
        </div>
        <div class="col-md-7">
            <h2>Напишите нам</h2>
            <form>
                <div class="mb-3">
                    <label for="name" class="form-label">Ваше имя</label>
                    <input type="text" class="form-control" id="name" placeholder="Иван Иванов">
                </div>
                <div class="mb-3">
                    <label for="email" class="form-label">Email</label>
                    <input type="email" class="form-control" id="email" placeholder="name@example.com">
                </div>
                <div class="mb-3">
                    <label for="message" class="form-label">Сообщение</label>
                    <textarea class="form-control" id="message" rows="4" placeholder="Ваш вопрос..."></textarea>
                </div>
                <button type="submit" class="btn btn-primary">Отправить</button>
            </form>
        </div>
    </div>
    <!-- Карта-заглушка (можно вставить iframe карты, но достаточно блока) -->
    <div class="mt-4 p-3 bg-light text-center rounded">
        <p>📍 Как нас найти: (здесь могла быть карта)</p>
    </div>
</div>
```

**Пояснения:**
- `mb-3` – отступ снизу для каждого поля формы.
- `form-control` – стилизованное поле ввода.
- `btn-primary` – синяя кнопка.

##### **Шаг 5. Проверка и доработка**

1. Убедитесь, что все файлы сохранены с кодировкой UTF-8.
2. Откройте `index.html` в браузере. Проверьте навигацию (переходы на `about.html` и `contacts.html`).
3. Нажмите F12 (Инструменты разработчика) и выберите режим эмуляции мобильного устройства (иконка телефона) – убедитесь, что сетка и навигация адаптируются.
4. Если изображения не отображаются, проверьте пути. Поместите изображения в папку `images` рядом с HTML-файлами, или используйте полные URL-адреса из интернета.
5. Если вы хотите использовать свои стили, добавьте их в тег `<style>` в `<head>` или создайте отдельный файл `style.css` и подключите его через `<link>` после Bootstrap (чтобы переопределить классы при необходимости).
